# Data Science Practicum Projects (DS_Practicum_Projects)

В данном репозитории собраны проекты по Data Science, выполненные в ходе обучения и практики.
Каждый проект посвящён отдельной теме и демонстрирует применение методов анализа данных и машинного обучения.

## Проекты

- **Линейные модели**  
  Регрессия и классификация, анализ признаков, оценка качества моделей.

- **Обучение с учителем**  
  Классические алгоритмы supervised learning и сравнение их качества.

- **Машинное обучение для текстов**  
  Предобработка текстов, векторизация и базовые NLP-модели.

- **Анализ временных рядов**  
  Исследование временных рядов, анализ трендов и прогнозирование.

- **Численные методы**  
  Методы оптимизации и численные решения задач.

- **Машинное обучение в бизнесе**  
  Прикладные задачи машинного обучения с фокусом на бизнес-метрики.

## Назначение репозитория

Репозиторий является частью портфолио и отражает практический опыт работы с данными и моделями машинного обучения.
